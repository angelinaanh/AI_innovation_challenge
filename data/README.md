# Data Module

## Purpose

Stores sample and processed demo data.

## Inputs

- Scratch course seed data.
- Sample student profiles.
- Evaluation examples.

## Outputs

- Demo data fixtures.
- Processed data used by local scripts or seeds.

## Dependencies

- Database seed scripts.
- AI evaluation workflows.

## Limitations

- No real child PII.
- Use synthetic demo data only unless explicit consent and privacy handling exist.
