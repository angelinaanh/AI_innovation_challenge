# Presentation Module

## Purpose

Pitch deck assets, demo script, screenshots, and video planning.

## Inputs

- Product story.
- Architecture diagrams.
- Demo screenshots.

## Outputs

- Pitch deck.
- Demo script.
- Judge FAQ.
- Backup demo assets.

## Dependencies

- `docs/pitch`.
- Running MVP screenshots.

## Limitations

- Presentation should sell the transformation before technical details.
