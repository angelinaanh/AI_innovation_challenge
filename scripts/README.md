# Scripts Module

## Purpose

Automation scripts for setup, seed, test, export, and demo preparation.

## Inputs

- Local environment variables.
- Database connection settings.
- Source data files.

## Outputs

- Seeded database.
- Generated demo data.
- QA reports.

## Dependencies

- Node.js runtime.
- Supabase CLI or API clients when added.

## Limitations

- Scripts must fail loudly on missing env.
- Scripts must not print secret values.
